CSS Selectors

Element Selector

  • element{...}
    • p{...}
    • h1{...}

    • class Selector

  • .class{...}
    • <div class="card" ><div/>
    • .card{...}

    • id Selector

  • #id{...}
    • <div id="main-section" ><div/>
    • #main-section{...}

    • Attribute Selector

  • [attribute='value']{...}
    • <input type="password" />
    • [type:"password"]{...}

    • Universal Selector

  • *{...}
    • Selects all elements in the document.
