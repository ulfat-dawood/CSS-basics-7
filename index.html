CSS Selectors

Element Selector

  • element{...}
    • p{...}
    • h1{...}

    • class Selector

  • .class{...}
    • <div class="card" ><div/>
    • .card{...}

    • id Selector

  • #id{...}
    • <div id="main-section" ><div/>
    • #main-section{...}

    • Attribute Selector

  • [attribute='value']{...}
    • <input type="password" />
    • [type:"password"]{...}

    • Universal Selector

  • *{...}
    • Selects all elements in the document.

Chaining

element.class

  • div.btn{...}
  • select all <div> elemnts with the class .btn

element#id

  • div#main{...}
  • select the <div> element having id #main

element[attribute='value']

  • input[type='text']{...}
  • select all <input> elements with type=text

.class.class

  • .btn.warning{...}
  • select all elemnts having both classes .btn and .warning

.class#id

  • .btn#signup{...}
  • select the element having the class .btn and id #signup

Combinators

selector ‚éµ selector

  • ul li{...}
  • Within ul, select all il.
  • div .card{...}
  • Within div, select all elements with the class .cards (not only direct children elements).
  • body table td{...}
  • Select all cells within tables inside body.
  • header nav .btn{...}
  • select all elements with the class .btn within nav that's inside header.

Selecto > Selector

  • div > h1{...}
  • Select all <h1> elements that are direct children of <div>.
  • div.article > p{..}
  • Select all <p> elements that are direct children of <div> with class .article

Selector + Selector

  • h1 + p{...}
  • Selects the next immediate sibling to h1.
  • article 1 is selected

Selector ~ Selector

  • h1 ~ p{...}
  • Selects all siblings after h1.
  • article 1 and article 2 are selected

:nth-child(nth)

  • where (nth): odd | even | (order) | (hops n +order)

(odd)

  • :nth-child(odd)
  • every element whose index is odd

(even)

  • :nth-child(even)
  • every elememt whose index is even

(order)

  • :nth-child(4)
  • 4th element

(hops n +order)

  • :nth-child(2n+1)
  • every second element (2 hopes) after the 1st element
